# 御池生成AIスクール ランディングページ アプリ概要

## 概要

御池生成AIスクールの受講申込を促すランディングページ。  
LINE公式アカウントへの友だち追加（クーポン取得）をゴールとするシングルページ構成。

## 技術スタック

- HTML / Tailwind CSS（CDN）
- Lucide Icons（CDN）
- Google Fonts（Noto Sans JP）
- フレームワーク不使用・静的ファイル1枚構成

## ページ構成

| セクション | 内容 |
|---|---|
| Header | ロゴ + 「限定クーポンGET」ボタン（LINE URL） |
| Hero | キャッチコピー + 入門コース割引ボタン（LINE URL） |
| Problem | 課題提示（手作業の悩み2点） |
| Benefits | 御池で身につく3つの力 |
| Courses | 入門コース（少人数制） / 個別プライベートレッスン |
| Team | 講師プロフィール（小川英幸） + パソコープ烏丸御池 |
| CTA | LINE公式アカウント友だち追加ボタン |
| Footer | 住所・連絡先・コピーライト |

## CTA（LINE公式）URL

`https://lin.ee/hJfLfscS`

CTAボタンが設置されている箇所：
- ヘッダー「限定クーポンGET」
- ヒーロー「入門コース 33,000円 → 22,000円」
- CTAセクション「友だち追加してクーポンGET」
